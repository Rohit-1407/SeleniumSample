Steps to execute project : 

1. Start Eclipse IDE

2. Pull the project from GIT Repo (public Repo)
	- CLick import -> GIT -> URL to the project (https://github.com/Rohit-1407/SeleniumSample/tree/master/JitterBit-CodingTest)

3. Execute the project
	- Update the chromedriver.exe location in the testSuite.java (Line # 17)
	- Run as -> TestNG class

4. All the test cases should then execute and report will be generated in the console and test-output folder
